🍽️ Cantina 67

Sistema web desenvolvido em Django para gerenciamento de uma cantina, incluindo controle de clientes, pedidos, produtos e pagamentos.

📌 Sobre o Projeto

O Cantina 67 é uma aplicação web que tem como objetivo facilitar a gestão de uma cantina, permitindo o cadastro de clientes, controle de pedidos, gerenciamento de produtos e registro de pagamentos.

🚀 Funcionalidades
Cadastro e gerenciamento de clientes
Controle de produtos
Criação e acompanhamento de pedidos
Registro e controle de pagamentos
Integração entre os módulos do sistema
🧩 Aplicativos do Projeto

O projeto está dividido nos seguintes apps Django:

👤 clientes

Responsável pelo gerenciamento de clientes:

Cadastro de clientes
Atualização de informações
Listagem e consulta
💳 pagamentos

Gerencia as transações financeiras:

Registro de pagamentos
Histórico de transações
Associação com pedidos
🛒 pedidos

Controle de pedidos realizados:

Criação de pedidos
Itens do pedido
Status do pedido
📦 produtos

Responsável pelo controle de produtos:

Cadastro de produtos
Controle de estoque
Preços e disponibilidade
🛠️ Tecnologias Utilizadas
Python 3.13
Django
SQLite (ou outro banco configurado)
HTML, CSS, JavaScript (para o frontend)
